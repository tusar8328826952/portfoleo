## 🚀 How to Use

### 1. **Add Your Profile Photo**
- Save your profile photo as `profile.jpg` in the portfolio folder
- Or rename your image to `profile.jpg` and place it in the same directory as `index.html`

### 2. **Update Contact Information**
Open `index.html` and update the Contact section with your actual details:
- Email: Replace `your.email@example.com`
- Phone: Replace `+91-XXXXXXXXXX`
- Add your social media links (optional)

### 3. **Open in Browser**
Simply open `index.html` in your web browser to view your portfolio!

## 🎨 Customization

### Colors
The portfolio uses a purple gradient color scheme. To change colors:
- Find `#667eea` (primary color) and `#764ba2` (secondary color) in the CSS
- Replace with your preferred colors

### Fonts
The site uses 'Segoe UI' as default. You can change the font in the CSS:
```css
body {
    font-family: 'Your Font Name', sans-serif;
}
```

### Content
All text content can be edited directly in the HTML file. Update:
- Your name, bio, and skills
- Project descriptions and technologies
- Contact information
